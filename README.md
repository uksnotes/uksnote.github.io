### 🌟 Core Projects

#### 1. 2Q PASS (우량 고객 자동 승인 시스템)
- **내용:** 370만 고객 데이터를 클러스터링하여 19개 우량 고객군 정의 및 자동 승인 룰 설계
- **성과:** 전체 체결 계약의 50.5% 이상 유입, 국내 특허 출원 및 언론 보도

#### 2. 올바로 모델 (질병 고지 자동화)
- **내용:** 보험 청구 및 가입 데이터를 분석해 고지 필요 질병을 자동 분류하는 AI 모델
- **성과:** 심사 정확도 90.5%, 평균 심사 시간 단축(90초 → 70초), 월 3.5만 건 자동화

#### 3. 올바로 챗봇 (보험 약관 AI 챗봇)
- **내용:** LangGraph 기반 멀티소스 RAG 시스템으로 보험 약관 및 고객 DB 질의응답 구현
- **성과:** 반복 질문 처리 시간 60% 단축 목표(PoC 완료)
